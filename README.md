# Crash Out

A simple, private, and high-intensity motivation utility. Built to help users "release the pressure" and get back to the grind.

## Features
- **Pressure Release:** A kinetic "venting" box that destroys stress-text on release.
- **Daily Intensity:** Motivational quotes focused on grit and resilience.
- **Privacy First:** No data collection, no ads, and no logs. Everything happens locally.
- **Haptic Feedback:** Built-in vibration for a physical release feel (Mobile only).

## Tech Stack
- HTML5 / CSS3
- JavaScript
- PWA (Progressive Web App) manifest for mobile installation.
